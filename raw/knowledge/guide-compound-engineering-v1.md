# Compound Engineering 工作流指南

> 从 Every Inc. 导入的 AI 驱动开发工作流
> 来源: https://github.com/EveryInc/compound-engineering-plugin

## 核心工作流

### CE 五阶段工作流

```
Brainstorm → Plan → Work → Review → Compound
```

1. **Brainstorm (头脑风暴)** - 探索需求和方案
2. **Plan (规划)** - 制定详细实施计划
3. **Work (执行)** - 系统化执行计划
4. **Review (审查)** - 多 Agent 代码审查
5. **Compound (复合)** - 记录经验教训
