# 技能与工具开发类技能

## 🔧 分类描述
本目录包含MCP工具开发、技能创建、内存系统、上下文管理等相关的技能，主要用于技能开发、工具构建、系统架构等工作。

## 📊 技能列表

### 🔨 工具开发
- `mcp-builder` - MCP构建工具
- `tool-design` - 工具设计
- `memory-systems` - 内存系统
- `context-fundamentals` - 上下文基础

### 🧠 技能创建与维护
- `creating-skills` - 技能创建
- `skill-creator` - 技能创建工具
- `skill-seekers` - 技能搜索器
- `skills-updater` - 技能更新工具

### 📚 技能管理工具
- `skills-updater` - 技能更新工具

## 📖 使用指南
1. 对于工具开发：使用 `mcp-builder` 和 `tool-design`
2. 对于技能创建：使用 `creating-skills` 和 `skill-creator`
3. 对于技能管理：使用 `skill-seekers` 和 `skills-updater`
4. 对于系统架构：使用 `memory-systems` 和 `context-fundamentals`

## 🚀 推荐学习路径
1. 先学习 `skill-creator` 掌握技能创建基础
2. 然后学习 `creating-skills` 掌握完整技能开发流程
3. 接着学习 `mcp-builder` 掌握MCP工具开发
4. 深入学习 `memory-systems` 掌握内存系统架构
5. 最后学习 `context-fundamentals` 掌握上下文管理
