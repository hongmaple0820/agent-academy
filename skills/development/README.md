# 开发与编程类技能

## 💻 分类描述
本目录包含软件开发、编程工具、代码审查、项目开发流程相关的技能，涵盖从需求分析到部署的完整开发生命周期。

## 📊 技能列表

### 🔧 核心开发流程
- `programming-workflow` - 编程工作流（独立核心技能）

### 🛠️ 开发工具
- `coding-agent` - 编程助手
- `git-commit` - Git提交管理
- `writing-github-actions` - GitHub Actions编写

### 🔍 代码审查与测试
- `requesting-code-review` - 代码审查请求
- `receiving-code-review` - 代码审查接收
- `test-driven-development` - 测试驱动开发
- `testing-strategies` - 测试策略
- `property-based-testing` - 基于属性的测试
- `root-cause-tracing` - 根原因追踪
- `tdd` - 测试驱动开发（简版）

### 🚀 部署与项目管理
- `deploying-applications` - 应用部署
- `finishing-a-development-branch` - 开发分支完成
- `systematic-debugging` - 系统化调试
- `verification-before-completion` - 完成前验证



## 📖 使用指南
1. 对于日常开发：使用 `programming-workflow` 作为核心工作流程
2. 对于代码审查：使用 `requesting-code-review` 和 `receiving-code-review`
3. 对于测试开发：使用 `test-driven-development` 和 `testing-strategies`
4. 对于部署管理：使用 `deploying-applications`
5. 对于Git管理：使用 `git-commit`

## 🚀 推荐学习路径
1. 先学习 `programming-workflow`，建立核心开发流程
2. 然后学习 `coding-agent`，掌握编程助手的使用
3. 接着学习 `git-commit`，掌握Git提交管理
4. 深入学习 `test-driven-development`，掌握测试驱动开发方法
5. 最后学习 `deploying-applications`，掌握应用部署技能
