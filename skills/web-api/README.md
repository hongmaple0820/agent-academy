## 一、分类说明

本分类专注于Web开发和API设计相关技能，包括：

- **Web开发**：前端页面、响应式设计、SEO优化
- **API设计**：RESTful API、GraphQL、API文档
- **表单构建**：表单验证、表单组件、动态表单
- **表格构建**：数据表格、分页排序、数据展示
- **Web组件**：UI组件库、自定义组件

## 四、技能分类标准

### 4.1 归类标准

符合以下任一条件的技能应归类到本分类：

1. **关键词匹配**：
   - web, website, webpage
   - api, rest, graphql
   - form, table, input
   - http, request, response

2. **功能匹配**：
   - 构建Web页面或组件
   - 设计或调用API
   - 创建表单或表格
   - 处理HTTP请求

### 4.2 排除标准

以下情况不应归类到本分类：

- 纯前端框架技能 → `frameworks`
- UI/UX设计技能 → `design`
- 安全测试技能 → `security-testing`
