## 🔍 快速查找指南

### 按功能查找
1. **需要项目管理工具** → 查看 `pm-product/` 目录
2. **需要开发工具** → 查看 `development/` 目录
3. **需要设计技能** → 查看 `design/` 目录
4. **需要文档写作** → 查看 `documentation/` 目录
5. **需要数据分析** → 查看 `data-analysis/` 目录

### 按使用场景查找
1. **日常复盘** → 直接使用 `daily-review/`
2. **编程开发** → 直接使用 `programming-workflow/`
3. **产品管理** → 直接使用 `pm-skills/`
4. **项目规范** → 直接使用 `project-standards/`

## 🚀 最佳实践
1. **优先使用核心技能**：高频核心技能使用更频繁，功能更完善
2. **按类别查找**：根据任务类型查找对应分类目录
3. **交叉使用**：多个分类的技能可以组合使用
4. **定期更新**：技能库会定期更新和优化
