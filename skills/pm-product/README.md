# PM与产品管理类技能

## 📋 分类描述
本目录包含产品管理、项目管理相关的技能，主要用于产品规划、需求分析、团队协作和项目管理相关工作。

## 📊 技能列表

### 🎯 PM技能库
- `pm-skills` - PM产品经理技能包（65个技能）

### 📋 项目管理工具
- `product-team` - 产品团队协作
- `project-management` - 项目管理
- `project-standards` - 项目规范指南（独立核心技能）

### 🧠 规划与执行
- `brainstorming` - 结构化头脑风暴
- `planning-with-files` - 基于文件的计划管理
- `executing-plans` - 计划执行

### 📁 已迁移技能目录

## 📖 使用指南
1. 对于产品需求分析：使用 `pm-skills` 中的需求管理技能
2. 对于项目规划：使用 `project-standards` 中的标准指南
3. 对于团队协作：使用 `product-team` 中的协作技能
4. 对于项目管理：使用 `project-management` 中的管理工具

## 🚀 推荐学习路径
1. 先学习 `project-standards`，了解项目管理的基本规范
2. 然后学习 `pm-skills`，掌握产品经理的核心技能
3. 接着学习 `product-team` 和 `project-management`，提高团队协作和项目管理能力
4. 最后学习 `brainstorming` 和 `planning-with-files`，提升需求分析和规划能力
