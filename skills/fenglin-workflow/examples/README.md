## 如何选择参考案例

### 根据项目复杂度

| 你的项目 | 参考案例 |
|----------|----------|
| 🟢 简单 (< 2小时) | Chat-Hub 的简化部分 |
| 🟡 中等 (2小时-2天) | Chat-Hub |
| 🔴 复杂 (> 2天) | ChessVerse, OpenClaw Zero Token |

### 根据项目类型

| 项目类型 | 参考案例 |
|----------|----------|
| Web 应用 | ChessVerse |
| 开发者工具 | OpenClaw Zero Token |
| 消息/中间件 | Chat-Hub |
| AI 集成 | ChessVerse, OpenClaw Zero Token |

## 使用建议

1. **先读 SKILL.md**: 理解枫林工作流的基本概念
2. **选择对应案例**: 根据你的项目复杂度选择参考案例
3. **关注决策过程**: 学习案例中的关键决策点
4. **复用经验教训**: 避免案例中提到的问题
5. **使用对应模板**: 配合 [templates/](../templates/) 中的模板使用
