# 文档与写作类技能

## 📝 分类描述
本目录包含文档写作、知识管理、格式转换、图表生成相关的技能，主要用于文档创作、内容管理、格式转换等工作。

## 📊 技能列表

### 📝 核心写作技能
- `daily-review` - 每日复盘技能（独立核心技能）
- `writing-skills` - 写作技能
- `writing-plans` - 写作计划

### 📖 文档工具
- `markdown-to-epub` - Markdown转EPUB
- `changelog-generator` - 变更日志生成器
- `diagram-generator` - 图表生成器

### 📚 知识管理
- `knowledge-base-maintenance` - 知识库维护

## 📖 使用指南
1. 对于每日复盘：使用 `daily-review`
2. 对于文档写作：使用 `writing-skills` 和 `writing-plans`
3. 对于格式转换：使用 `markdown-to-epub`
4. 对于变更记录：使用 `changelog-generator`
5. 对于图表生成：使用 `diagram-generator`

## 🚀 推荐学习路径
1. 先学习 `daily-review` 建立每日复盘习惯
2. 然后学习 `writing-skills` 提升写作能力
3. 接着学习 `writing-plans` 掌握写作计划技能
4. 深入学习 `markdown-to-epub` 掌握格式转换技能
5. 最后学习 `changelog-generator` 掌握变更日志生成技能
