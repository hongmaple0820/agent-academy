# AI与机器学习类技能

## 🤖 分类描述
本目录包含机器学习模型训练、数据集管理、评估工具、HuggingFace相关技能，主要用于AI/ML相关工作。

## 📊 技能列表

### 🤖 HuggingFace工具套件
- `hugging-face-model-trainer` - HuggingFace模型训练
- `hugging-face-datasets` - HuggingFace数据集
- `hugging-face-evaluation` - HuggingFace评估
- `hugging-face-cli` - HuggingFace命令行工具
- `hugging-face-trackio` - HuggingFace跟踪工具

### 📚 深度学习框架
- `pytorch-lightning` - PyTorch Lightning框架

### 🔍 AI相关技能
- 未来可能添加更多AI/ML相关技能

## 📖 使用指南
1. 对于模型训练：使用 `hugging-face-model-trainer`
2. 对于数据处理：使用 `hugging-face-datasets`
3. 对于模型评估：使用 `hugging-face-evaluation`
4. 对于命令行操作：使用 `hugging-face-cli`
5. 对于深度学习：使用 `pytorch-lightning`

## 🚀 推荐学习路径
1. 先学习 `hugging-face-datasets` 掌握数据集管理
2. 然后学习 `hugging-face-model-trainer` 掌握模型训练
3. 接着学习 `hugging-face-evaluation` 掌握模型评估
4. 深入学习 `hugging-face-cli` 掌握命令行工具
5. 最后学习 `pytorch-lightning` 掌握深度学习框架
