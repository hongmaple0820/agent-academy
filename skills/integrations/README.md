# 存储与集成类技能

## 🔗 分类描述
本目录包含数据库、云服务、Google服务、第三方平台集成的技能，主要用于数据存储、云集成、平台集成等工作。

## 📊 技能列表

### 🔗 数据库与存储
- `supabase-postgres-best-practices` - Supabase PostgreSQL最佳实践
- `secret-management` - 密钥管理

### 🔗 Google服务集成
- `google-calendar` - Google日历集成
- `google-docs` - Google文档集成
- `google-drive` - Google云端硬盘集成
- `google-sheets` - Google Sheets集成

### 🔗 第三方平台集成
- `notion` - Notion集成
- `slack` - Slack集成
- `discord` - Discord集成

### 🔗 其他集成服务
- 未来可能添加更多集成技能

## 📖 使用指南
1. 对于数据库管理：使用 `supabase-postgres-best-practices`
2. 对于密钥管理：使用 `secret-management`
3. 对于Google服务：使用相应的Google集成技能
4. 对于平台集成：使用 `notion`、`slack`、`discord`

## 🚀 推荐学习路径
1. 先学习 `secret-management` 掌握密钥管理基础
2. 然后学习 `supabase-postgres-best-practices` 掌握数据库管理
3. 接着学习Google服务集成技能
4. 最后学习第三方平台集成技能
