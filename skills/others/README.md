### 📚 开发工具与技巧

## 📖 使用指南
1. 对于团队管理：使用 `engineering-team` 和 `c-level-advisor`
2. 对于营销推广：使用 `marketing-skill`
3. 对于Git管理：使用 `using-git-worktrees`
4. 对于开发技巧：使用 `using-superpowers`
5. 对于网页设计：使用 `web-design-guidelines`

## 🚀 推荐学习路径
1. 先学习 `using-git-worktrees` 掌握高级Git技能
2. 然后学习 `using-superpowers` 掌握高效开发技巧
3. 接着学习 `engineering-team` 掌握团队管理
4. 深入学习 `c-level-advisor` 掌握高级顾问技能
5. 最后学习 `marketing-skill` 掌握营销技能
