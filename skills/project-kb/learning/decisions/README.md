# 决策记录 (ADR)

> Architecture Decision Records - 记录重要的架构和技术决策。

---

## 📝 ADR 格式

每个决策使用以下格式：

```markdown
# ADR-XXX: 决策标题

## 状态
- 提议 / 已接受 / 已弃用 / 已替代

## 背景
描述需要做出决策的背景和上下文。

## 决策
明确描述做出的决策。

## 原因
解释为什么选择这个方案。

## 后果
- 正面影响
- 负面影响
- 技术债务（如有）

## 替代方案
列出考虑过的其他方案及不选的原因。

## 参考
相关链接和参考资料。

---
*决策日期: YYYY-MM-DD*
*决策人: 姓名*
```
