# 最佳实践 (Best Practices)

> 记录项目中的最佳实践，提升代码质量和开发效率。

---

## 📝 使用说明

每个最佳实践应包含：
- **场景** - 适用什么场景
- **实践** - 具体做法
- **示例** - 代码示例
- **收益** - 带来什么好处
- **注意事项** - 需要注意什么

---

## 📂 目录

按技术栈分类：

- `javascript/` - JavaScript/TypeScript 最佳实践
- `python/` - Python 最佳实践
- `go/` - Go 最佳实践
- `database/` - 数据库最佳实践
- `architecture/` - 架构设计最佳实践
- `testing/` - 测试最佳实践
- `deployment/` - 部署运维最佳实践

---

## 🏷️ 标签

| 标签 | 含义 | 颜色 |
|------|------|------|
| `essential` | 强烈推荐 | 🔴 |
| `recommended` | 建议采用 | 🟡 |
| `optional` | 可选 | 🟢 |
