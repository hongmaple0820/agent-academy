# 数据分析与研究类技能

## 📊 分类描述
本目录包含数据分析、可视化、机器学习、统计分析相关的技能，主要用于数据研究、统计分析、可视化展示等工作。

## 📊 技能列表

### 🔬 数据分析基础
- `deep-research` - 深入研究
- `statistical-analysis` - 统计分析
- `csv-analyzer` - CSV数据分析器

### 📈 数据可视化
- `visualizing-data` - 数据可视化
- `matplotlib` - Matplotlib图表库
- `d3js-visualization` - D3.js可视化

### 🔍 搜索与过滤
- `implementing-search-filter` - 搜索过滤器实现

### 📚 数据科学工具
- 未来可能添加机器学习、数据挖掘等技能

## 📖 使用指南
1. 对于数据分析：使用 `statistical-analysis` 和 `csv-analyzer`
2. 对于数据可视化：使用 `visualizing-data` 和 `matplotlib`
3. 对于Web可视化：使用 `d3js-visualization`
4. 对于搜索功能：使用 `implementing-search-filter`

## 🚀 推荐学习路径
1. 先学习 `statistical-analysis` 掌握统计分析基础
2. 然后学习 `csv-analyzer` 掌握CSV数据处理
3. 接着学习 `visualizing-data` 掌握数据可视化技能
4. 深入学习 `matplotlib` 掌握图表制作
5. 最后学习 `d3js-visualization` 掌握Web可视化
