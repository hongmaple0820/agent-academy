# 设计与界面类技能

## 🎨 分类描述
本目录包含用户界面设计、API设计、布局设计、Canvas设计相关的技能，主要用于前端设计、UI/UX、交互设计等工作。

## 📊 技能列表

### 🎨 UI设计工具
- `frontend-design` - 前端设计
- `designing-layouts` - 布局设计
- `designing-apis` - API设计
- `canvas` - Canvas设计

### 🖌️ 设计系统与组件
- `ui-ux-pro-max` - UI/UX高级设计
- `ppt-style-guide` - PPT样式指南
- `composition-patterns` - 组合模式

### 📐 表单与表格
- `building-forms` - 表单构建
- `building-tables` - 表格构建

### 🔐 安全合约设计
- `building-secure-contracts` - 安全合约构建

## 📖 使用指南
1. 对于前端设计：使用 `frontend-design` 和 `designing-layouts`
2. 对于API设计：使用 `designing-apis`
3. 对于Canvas设计：使用 `canvas`
4. 对于设计系统：使用 `ui-ux-pro-max` 和 `composition-patterns`
5. 对于表单和表格：使用 `building-forms` 和 `building-tables`

## 🚀 推荐学习路径
1. 先学习 `frontend-design` 掌握基础前端设计技能
2. 然后学习 `designing-layouts` 掌握布局设计
3. 接着学习 `designing-apis` 掌握API设计技能
4. 深入学习 `canvas` 掌握Canvas设计技术
5. 最后学习 `ui-ux-pro-max` 掌握高级UI/UX设计

## 📍 关联目录
- `../development/` - 开发与编程类技能
- `../frameworks/` - 框架与库类技能
- `../web-api/` - Web与API开发类技能
