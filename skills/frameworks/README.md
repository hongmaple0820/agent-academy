# 框架与库类技能

## 📚 分类描述
本目录包含React、React Native、组件库、组合模式相关的技能，主要用于前端框架、库、设计模式的开发与使用。

## 📊 技能列表

### 📚 React框架技能
- `react-best-practices` - React最佳实践
- `react-best-practices-build` - React构建最佳实践
- `react-native-skills` - React Native技能

### 🔧 组件设计模式
- `composition-patterns` - 组合模式设计

### 📚 其他框架
- 未来可能添加更多框架技能

## 📖 使用指南
1. 对于React开发：使用 `react-best-practices`
2. 对于React构建：使用 `react-best-practices-build`
3. 对于React Native：使用 `react-native-skills`
4. 对于组件设计：使用 `composition-patterns`

## 🚀 推荐学习路径
1. 先学习 `react-best-practices` 掌握React基础
2. 然后学习 `react-best-practices-build` 掌握构建最佳实践
3. 接着学习 `composition-patterns` 掌握组件设计模式
4. 最后学习 `react-native-skills` 掌握移动端开发

## 📍 关联目录
- `../design/` - 设计与界面类技能
- `../development/` - 开发与编程类技能
- `../web-api/` - Web与API开发类技能
