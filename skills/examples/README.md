## 🎓 如何使用示例

### 1. 学习阶段
1. 阅读示例文件，了解完整流程
2. 理解每个步骤的目的和输出
3. 注意技能之间的组合使用

### 2. 实践阶段
1. 按照示例步骤操作
2. 根据实际情况调整参数
3. 记录自己的使用心得

### 3. 进阶阶段
1. 组合多个技能完成复杂任务
2. 创建自己的使用模式
3. 贡献新的示例到本库

## 📝 示例结构

每个示例包含：
```
示例名称/
├── README.md           # 示例说明
├── session.md          # 完整对话记录
├── outputs/            # 生成的文件示例
│   ├── task_plan.md
│   └── findings.md
└── tips.md            # 使用技巧和注意事项
```

## 🔄 贡献示例
