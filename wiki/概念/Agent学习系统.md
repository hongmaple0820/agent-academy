# Agent 学习系统

三层知识库架构，让 AI Agent 能够从错误中学习、复用经验、自主决策和自我进化。

## 三层架构

| 层级 | 位置 | 内容 | 频率 |
|------|------|------|------|
| **热层** (实时) | `memory/daily/YYYY-MM-DD.md` | 当天错误、解决方案、临时经验 | 实时 |
| **温层** (经验) | `learnings/` | 结构化学习总结 | 按事件 |
| **冷层** (知识) | `knowledge/` | 可复用的指南、模式、架构 | 定期 |

## 五种能力

1. **错误学习** — 记录错误和解决方案，避免重复踩坑
2. **经验复用** — 利用已有代码、工具、技能
3. **资源发现** — 知道有什么可用，避免重复造轮子
4. **自主决策** — 少问多做，主动解决问题
5. **自我进化** — 验证 → 反思 → 总结 → 规划
