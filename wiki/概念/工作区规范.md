# 工作区规范

Agent 工作区的标准文件结构和设计约定。

## 标准文件

| 文件 | 用途 |
|------|------|
| `SOUL.md` | AI 人格定义 |
| `AGENTS.md` | 行为规范和约束 |
| `USER.md` | 用户信息和偏好 |
| `MEMORY.md` | 长期记忆入口 |
| `HEARTBEAT.md` | 心跳/定时任务 |
| `memory/` | 每日工作日志 |
| `learnings/` | 学习总结 |

## 设计原则

1. **约定优于配置** — 标准文件名，标准位置
2. **Git 管理** — 全部版本控制，可分支和审查
3. **Agent 可读** — 清晰结构，让 Agent 能自主导航
4. **分层知识** — 热/温/冷三层，按访问频率
