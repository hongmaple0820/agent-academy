# 多 Agent 协作

Agent Academy 的多 Agent 协作模式 — 让多个 AI Agent 分工合作，完成复杂任务。

## 协作模式

| 模式 | 适用场景 | 说明 |
|------|----------|------|
| **主从模式** | 复杂任务分解 | 主 Agent 负责任务拆解，子 Agent 执行具体步骤 |
| **流水线模式** | 多阶段流程 | 每个 Agent 负责一个环节，按顺序传递 |
| **竞争模式** | 多方案探索 | 多个 Agent 并行探索不同方案，择优录取 |
| **协作模式** | 大型项目 | 多个 Agent 各司其职，协同完成 |

## 常见角色

- **Planner** — 负责规划和任务拆解
- **Executor** — 执行具体的编码或操作任务
- **Reviewer** — 审查代码质量和方案可行性
- **Researcher** — 负责资料搜索和技术调研
- **Tester** — 负责测试验证和质量保障

## 协作协议

1. **任务分配** — 明确每个 Agent 的职责和边界
2. **信息传递** — 通过共享工作区和文件传递上下文
3. **结果验证** — 每个环节完成后需要验证
4. **冲突解决** — 出现分歧时由 Planner 决策
