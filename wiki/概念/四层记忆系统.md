# 四层记忆系统

Agent Academy 的记忆架构 — 四层设计让 AI 在不同时间尺度上保持上下文。

## 架构

| 层级 | 时间尺度 | 内容 | 存储 |
|------|----------|------|------|
| **会话记忆** | 当前对话 | 当前 session 的上下文 | 内存 |
| **日记记忆** | 天 | 每天的工作日志 | `memory/YYYY-MM-DD.md` |
| **经验记忆** | 周/月 | 结构化学习总结 | `learnings/` |
| **知识记忆** | 长期 | 可复用的指南和模式 | `knowledge/`、`skills/` |

## 关键组件

### QMD 本地语义搜索
- 本地搜索引擎，混合 BM25 + 向量搜索 + LLM 重排序
- 全部在设备端运行，节省 ~90% Token
- 同时提供 CLI 和 MCP Server

### 记忆管理策略
- 每日自动记录关键决策和错误
- 定期从日记中提炼经验到温层
- 从经验中抽象出可复用的知识到冷层
