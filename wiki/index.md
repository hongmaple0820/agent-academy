# Agent Academy Wiki

Agent Academy 的结构化知识库 — Karpathy LLM Wiki 模式。

## 架构

```
agent-academy/
├── wiki/                    # 结构化知识库
│   ├── index.md             # 本页，知识索引
│   ├── 实体/                 # 具体的人、项目、组织
│   ├── 概念/                 # 核心概念和原理
│   └── 主题/                 # 综合性主题
├── raw/                     # 原始资料层
│   ├── knowledge/           # 原始知识文档
│   ├── learnings/           # 原始学习记录
│   └── memory/              # 原始记忆文件
├── logs/                    # 操作日志
│   └── log.md
├── skills/                  # 800+ 技能库
├── templates/               # 工作区模板
└── scripts/                 # 安装部署脚本
```
