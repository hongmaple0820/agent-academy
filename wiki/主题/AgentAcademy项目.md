# Agent Academy 项目

开源 Agent 训练知识库 — 给 AI 助手办一张"终身学习卡"。

## 项目定位

- **目标**：提供系统化的知识、技能和规范，让 AI 助手真正能用、好用
- **规模**：800+ 精选技能 · 四层记忆系统 · 多 Agent 协作 · MCP 全家桶
- **开源**：MIT License，托管于 Gitee、GitHub、GitCode 三个平台

## 快速安装

```bash
# 一键安装（推荐）
curl -fsSL https://gitee.com/hongmaple/agent-academy/raw/master/scripts/install.sh | bash

# 手动安装
git clone https://gitee.com/hongmaple/agent-academy.git
cp -r agent-academy/skills/* ~/.agents/skills/
```

## 核心组件

- **`skills/`** — 800+ 技能库（核心价值）
- **`wiki/`** — 结构化知识库
- **`templates/`** — 工作区配置模板
- **`scripts/`** — 安装和部署脚本
- **`docs/`** — 文档中心
