> 开箱即用的 AI 协作技能库，让 AI 助手更强大

## ✨ 特性

- 📦 **800+ 精选技能** - 覆盖开发、设计、数据分析等领域
- 📚 **MCP 知识体系** - 完整的 Model Context Protocol 文档
- 🔄 **CI/CD 集成** - 自动化统计、验证、发布
- 🚀 **快速开始** - 一键安装，开箱即用

## 📦 安装

```bash
# 克隆仓库
git clone https://gitee.com/openclaw/ai-skill-hub.git

# 安装技能到你的 AI 助手
cp -r ai-skill-hub/skills/* ~/.agents/skills/
```

## 📂 目录结构

